simulate_many(self, num_iterations=100, num_gens=300, return_labeled=True, parallelize=True)

source code 

A helper method to call the simulate methods num_iterations times simulating num_gens generations each time, and then averaging the frequency of the resulting equilibria. Method calls are parallelized and attempt to use all available cores on the machine.

Parameters:
  • num_iterations (int) - the number of times to iterate the simulation
  • num_gens (int) - the number of generations to run each simulation witu
  • return_labeled (bool) - whether the distribution of classified equilibria that are returned should be labelled or simply listed with their keys inferred by their order
  • parallelize (bool) - whether or not to parallelize the computation, defaults to true, but an override when varying the parameters, as seen in the VariedGame class to achieve coarser parallelization
Returns: numpy.ndarray or dict
the frequency of time spent in each equilibria, defined by the game
