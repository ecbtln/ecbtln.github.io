Class NDimensionalData

source code

         |
        NDimensionalData
Known Subclasses:

A helper class that encapsulates varied multi-dimensional results. The results are stored as a recursive list of lists, whereby to get the simulation results for a given fixed set of parameters, we can iterate through the recursive list and index based on the indices of the independent parameter values desired.

marginalize(self, **kwargs)

Convert the N dimensional data to a lower-order dimension by iterating through the data and stitching together the appropriate subarrays. The keyword arguments are keyed by the keyword of the parameter, and their value is the value on which to marginalize the data (by fixing that parameter to that value, and therefore looking at one cross section of the higher order data
